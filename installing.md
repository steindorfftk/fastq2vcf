# sratools
- Run "sudo apt install sra-toolkit"
- After installation is complete, run "vdb-config -i"
- Use Tab to navigate to "Cache" and press enter.
- Navigate to "Choose" and set "fastq2vcf/temporary/fastq_dump" as the user-repository.
- Save and exit

# Fastqc
- Run "sudo apt install fastqc"

# Trimmomatic (Recommended: using conda)
- Run "conda install -c bioconda trimmomatic"
